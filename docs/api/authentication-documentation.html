POST /auth/login

Login

Authenticates a user and provides a JWT token.

Public ALL
Request Body
{
  "email": "admin@example.com",
  "password": "securePass123"
}
Example Response 200
{
  "success": true,
  "data": {
    "token": "eyJhbG...",
    "expiresIn": "8h"
  }
}
Assertions
  • res.status equals 200
  • res.body.success is true
  • Store res.body.data.token as {{authToken}}
GET /auth/profile

Get Profile

Retrieves the authenticated user's profile and permissions.

Bearer {{authToken}} ALL
Example Response 200
{
  "success": true,
  "data": {
    "employeeCode": "EMP-001",
    "firstName": "Admin",
    "lastName": "User",
    "roleCode": "ADMIN"
  }
}
Assertions
  • res.status equals 200
  • res.body.success is true
  • res.body.data contains employeeCode, roleCode
