Order Pipeline

Order CRUD with Mode A/B/C support

Integration Guide

Our system uses 3 creation modes: Mode A (Sender-to-Self), Mode B (Sender-to-Receivers), and Mode C (Combo). The backend automatically detects the mode based on whether 'products' (root) or 'receivers' (array) are provided. Testers should verify that each receiver correctly generates its own unique Parcel ID (PDS-XXX).

GET /orders?page=1&limit=20

List Orders

Paginated order list with derived status.

Authentication Bearer Token
Permissions ADMIN, OPERATOR, COURIER
Parameters
NameTypeDescription
page number Current page number
limit number Number of items per page
status string Filter by derived status (PENDING, DISPATCHED, etc.)
cURL Example
curl -X GET "{{base_url}}/orders?page=1&limit=20" \
  -H "Authorization: Bearer {{authToken}}" \

POST /orders

Create Order — Mode B (Sender → Receivers)

Creates an order with sender and multiple receivers. Each receiver generates a parcel.

💡
Developer Note: This is Mode B. Ensure each object in the 'receivers' array has its own 'products' array. The 'courierId' is mandatory to determine the shipping partner.
Authentication Bearer Token
Permissions ADMIN, OPERATOR
cURL Example
curl -X POST "{{base_url}}/orders" \
  -H "Authorization: Bearer {{authToken}}" \
  -H "Content-Type: application/json" \
  -d '{"senderName":"Ramesh Textiles","senderMobile":"9876543210","senderAddress":"14, Gandhi Nagar, Surat","courierId":1,"receivers":[{"receiverName":"Delhi Fabrics Ltd.","receiverPhone":"9123456780","address":"45, Karol Bagh","city":"New Delhi","state":"Delhi","pincode":"110005","products":[{"productId":1,"qty":5,"unitPrice":420}]}]}'
Request Body
{
  "senderName": "Ramesh Textiles",
  "senderMobile": "9876543210",
  "senderAddress": "14, Gandhi Nagar, Surat",
  "courierId": 1,
  "receivers": [
    {
      "receiverName": "Delhi Fabrics Ltd.",
      "receiverPhone": "9123456780",
      "address": "45, Karol Bagh",
      "city": "New Delhi",
      "state": "Delhi",
      "pincode": "110005",
      "products": [
        {
          "productId": 1,
          "qty": 5,
          "unitPrice": 420
        }
      ]
    }
  ]
}
Expected Status 201

PUT /orders/:id

Permissions ADMIN, OPERATOR
