Dispatch and Terminal States

Dispatch, deliver, and return parcel operations

POST /parcels/dispatch

Batch Dispatch

Dispatches one or more AWB_LINKED parcels.

Authentication Bearer Token
Permissions ADMIN, OPERATOR
cURL Example
curl -X POST "{{base_url}}/parcels/dispatch" \
  -H "Authorization: Bearer {{authToken}}" \
  -H "Content-Type: application/json" \
  -d '{"parcelDetailsIds":[1,2,3]}'
Request Body
{
  "parcelDetailsIds": [
    1,
    2,
    3
  ]
}
Expected Status 200
