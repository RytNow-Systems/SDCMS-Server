### Create Product
POST /api/v1/products
Content-Type: application/json
{
  "materialName": "Cotton Fiber",
  "materialRate": 500.50,
  "cuItemCode": "CF-001",
  "categoryId": 1,
  "unitId": 1,
  "materialDescription": "High quality cotton fiber"
}

### Get All Products
GET /api/v1/products?categoryId=1&unitId=1

### Get Product by ID
GET /api/v1/products/1

### Update Product
PUT /api/v1/products/1
Content-Type: application/json
{
  "materialName": "Premium Cotton Fiber",
  "materialRate": 550.00
}

### Delete Product
DELETE /api/v1/products/1
