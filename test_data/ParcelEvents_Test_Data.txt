### List System-Wide Events
Method: GET
URL: /api/v1/parcel-events?page=1&limit=50
Headers: { "Authorization": "Bearer {{authToken}}" }
Assertions:
- res.status equals 200
- res.body.data is Array
- res.body.meta.totalRows is Number

### Export Events CSV
Method: GET
URL: /api/v1/parcel-events/export
Headers: { "Authorization": "Bearer {{authToken}}" }
Assertions:
- res.status equals 200
- res.headers["content-type"] equals "text/csv"
