PARCEL EVENTS & AUDIT EXPORT — Test Data
Sprint 4 Feature C: Parcel Events & CSV Export
========================================

--- TEST 1: Browse All Events (No Filters) ---
Method: GET
URL: /api/v1/parcel-events?page=1&limit=50
Headers:
  Authorization: Bearer <OPERATOR_TOKEN>

Assertions:
  - Status: 200
  - Body: { success: true, data: [...], meta: { page, limit, totalRows, totalPages } }
  - Each event has: id, parcelId, orderCode, actionType, awbNumber, previousStatus, newStatus, scannedBy, timestamp

--- TEST 2: Browse Events with ActionType Filter ---
Method: GET
URL: /api/v1/parcel-events?actionType=AWB_LINK

  Authorization: Bearer <OPERATOR_TOKEN>

Headers:
  Authorization: Bearer <OPERATOR_TOKEN>

Assertions:
  - Status: 200
  - Response Header: Content-Type: text/csv
