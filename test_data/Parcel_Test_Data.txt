### List Parcels
Method: GET
URL: /api/v1/parcels?page=1&limit=20
Headers: { "Authorization": "Bearer {{authToken}}" }
Assertions:
- res.status equals 200
- res.body.success is true
- res.body.data is Array

### Get Parcel Details
Method: GET
URL: /api/v1/parcels/1
Headers: { "Authorization": "Bearer {{authToken}}" }
Assertions:
- res.status equals 200
- res.body.data.id equals 1
