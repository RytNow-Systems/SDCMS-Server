# Courier Test Data

## GET /api/v1/courier-partners
Method: GET
URL: /api/v1/courier-partners
Headers: { "Authorization": "Bearer {{authToken}}" }
Assertions:
1. res.status is 200
2. res.body.success is true
3. res.body.data[0].courierName exists
4. res.body.data[0].trackingUrlTemplate exists

## POST /api/v1/courier-partners
Method: POST
URL: /api/v1/courier-partners
Headers: { "Authorization": "Bearer {{authToken}}", "Content-Type": "application/json" }
Payload:
{
  "courierName": "Speedy Express",
  "phoneNo": "1234567890",
  "trackingUrlTemplate": "https://speedy.express/track?awb={AWB}"
}
Assertions:
1. res.status is 200
2. res.body.success is true
3. res.body.data.courierName is "Speedy Express"
