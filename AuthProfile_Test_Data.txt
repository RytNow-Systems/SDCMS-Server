### TEST 1: Get Profile (Valid Token)
**Description:** Fetches current user profile from DB via JWT.
**HTTP Method:** GET
**URL:** {{baseUrl}}/api/v1/auth/profile
**Headers:** 
- Authorization: Bearer {{token}}
- Content-Type: application/json

**Assertions:**
- Status Code: 200
- Success: true
- Body Path `data`: exists
- Body Path `data.EmployeeCode`: matches JWT identifier
- Body Path `data.FullName`: is string
- Body Path `data.EmailAddress`: is string
